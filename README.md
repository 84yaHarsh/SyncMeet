# SyncMeet

A modern Google Meet-inspired video conferencing SaaS built with React, Node.js, WebRTC, and Socket.IO.

## Getting Started

### Prerequisites
- Node.js (v20+)

### Setup

1. **Clone the repository**

2. **Server Setup**
   ```bash
   cd server
   npm install
   cp .env.example .env
   npm run dev
   ```

3. **Client Setup**
   ```bash
   cd client
   npm install
   cp .env.example .env
   npm run dev
   ```

## Features
- Google Login
- Real-time video/audio calling (WebRTC)
- Screen Sharing
- Meeting Chat
- Maximum 4 participants
